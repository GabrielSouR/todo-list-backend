# 📝 To-do List Backend

API simples e segura de gerenciamento de tarefas com autenticação de usuários, desenvolvida com Node.js e MySQL.

---

## 🚀 Tecnologias Utilizadas

- Node.js
- Express
- MySQL
- JWT (JSON Web Tokens)
- Bcrypt
- Express-validator
- Dotenv
- Nodemon
